\noindent \textbf{Question:} Given two vector fields
\begin{equation*}
\begin{split}
&\textbf{A}=(x+2y)\hat x+(x-z)\hat y+x\hat z\mathrm{~and}\\
&\textbf{B}=(1-z)\hat x+(z+2y)\hat y+2xz\hat z
\end{split}
\end{equation*}
a. Find the scalar component of \textbf{B} along \textbf{A} at (1, 2, -1).\\
b. Find the angle between \textbf{A} and \textbf{B} at (2, 1, 1).
